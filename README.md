# News-App

A simple list of news items.

## Features

- Pagination
- Pull to refresh
- MVVM architecture
- XCTests
- XCUITests

## Development

**Technologies**

- [Swift](https://swift.org/)
- [CocoaPods](https://cocoapods.org/)

**Requirements**

- Xcode 11 or higher
- iOS 13 or higher
